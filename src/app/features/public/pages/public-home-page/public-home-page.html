SISPARK

Plataforma para administrar parqueaderos con control operativo y trazabilidad.

Esta es la zona publica del proyecto. La operacion administrativa y el registro de vehiculos se realizan dentro del modulo seguro para usuarios autenticados.

Zona publica

Informacion general, acceso al sistema y futura vitrina de servicios para clientes y visitantes.

Area administrativa

Ingreso, salida, registro de vehiculos, control de pagos y operacion diaria.

Seguridad

Toda accion operativa queda asociada al usuario autenticado dentro del sistema.

Crecimiento

La base nueva ya esta pensada para clientes, vehiculos, mensualidades, pagos y auditoria.
